## Business Requirements

- Employees should be able to apply for leave
- Leave should go into Pending status
- Any approver should be able to approve
- Only the first approval counts
- Approved/Rejected requests cannot be changed
- Every action must be logged
- System must work with Gmail-based login

## Roles

- Employee
- Approver (Team Lead, HR, Manager)

## Rules

- A leave can be approved by any one approver
- No multi-level approval
- Approvers must see all pending leaves
- Employees must only see their own leaves
