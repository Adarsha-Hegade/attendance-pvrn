# Leave Approval System

A web-based internal tool for managing employee leave requests with role-based approvals.

This system allows:

- Employees to submit leave requests
- Multiple managers to act as approvers
- Any single approver to approve or reject a request
- Automatic locking after first decision
- Full audit trail

Built using:

- Next.js (App Router)
- React
- shadcn/ui
- Supabase (Auth + Database + Row Level Security)
